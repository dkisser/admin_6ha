Ø 注意事项：

1. 1024*600对话框，15条记录/页，居中。

2. 首先查询该用户是否是部门易耗品管理员：

select id from c_zcgl where appid=? and fzr=?。

如果为空，弹出对话框：您不是部门易耗品管理员，无法完成登记工作。

3. 查询V_LYHP视图，获得指定appId，操作部门（CZBM_ID）为上面id的操作列表。getBMLYHPList

l 操作部门（czbm_dept_name）

l 持有部门（dept_name）

l 操作数量（num）

4. 筛选：getBMLYHPList

l 操作人：搜索框，默认为空。WXUserController.queryWXUserUI

l 操作类型：下拉列表框，默认为空，包括全部。YHPCZLXService.getCZLXComboWithAll

l 易耗品类型：搜索框，默认为空。YHPLXController.queryYhplxUI。用户选择父类型，将该类型及其子类型对应的易耗品都找出来。
